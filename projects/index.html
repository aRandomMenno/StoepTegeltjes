HTML Comment Finder

This is an extension for Firefox based browsers that finds HTML comments on the current page. It was forked from the Chrome version by Astral and changed to work on Firefox based browsers. I improved the styles for the extension. But I made a security mistake by inputting the comments in the popup.html with `.innerHTML` instead of `.innerText`. This was caught by Astral and fixed by her.
