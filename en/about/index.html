Some facts about me!

I am Menno, a 16-year-old boy living in the Netherlands, near Rotterdam. I have a passion for programming and ethical hacking, although I must admit that I still need to improve, but I'm always learning new things. However, I often spend more time gaming than coding or learning to hack, and I'd like to change that. I also enjoy things that help me grow as a person and make the world a better place.

When I first started programming, it was in HTML. Some people don't consider that a programming language, and that might be true, but HTML introduced me to the concept of programming. Nowadays, I know a fair amount about HTML, CSS, JS, and Python.

Compared to other programmers, I might not know much, but I don't like comparing myself to others. Often, when I'm coding, I live by the rule: "My code doesn't work, I don't know why. I change something, now it works, but I still don't know why." I am trying more and more to understand the code I write myself or the code I copy.
