Welcome to the data Generator!

What is this?

On this page you can generate random strings of data. This data can be made of letters, numbers, specials characters or a combination of those. You can choose the amount of strings, the length of each string and a separator.

Or you can choose your own characters that the strings can be made of.
WARN: if you include '<' or '>' in the custom "dictionary" the result might display weirdly. this is because it then tries to display the html code for the characters.
