Het Grafisch Lyceum Rotterdam

Wie ben ik?

Ik ben Menno van Zoelen, een student aan het Grafisch Lyceum Rotterdam. Ik ben 16 jaar oud en zit in het eerste jaar van de opleiding Creative Software Developer. Ik ben namelijk dit jaar begonnen met de opleiding. De reden dat ik voor deze opleiding heb gekozen is omdat ik het leuk vind om te programmeren en ik wil graag beter leren programmeren.

Ik heb al redelijk wat ervaring met programmeren, in het specifiek HTML & CSS. Want ik heb mijn eigen website in HTML & CSS gemaakt. Ik heb ook ervaring met JavaScript, maar ik ben nog niet zo goed in JavaScript. Ik wil graag beter worden in JavaScript en andere programmeertalen zoals Rust, Python & Java.

De dingen waar ik in geinstresseerd ben zijn: programmeren, gamen, muziek luisteren, slapen.
