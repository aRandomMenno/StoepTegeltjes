Enkele feiten over mij!

Ik ben Menno, een 16-jarige jongen die in Nederland woont, nabij Rotterdam. Ik heb een passie voor programmeren en ethisch hacken, hoewel ik moet toegeven dat ik nog moet verbeteren, maar ik leer altijd nieuwe dingen. Echter, ik besteed vaak meer tijd aan gamen dan aan coderen of leren hacken, en ik zou dat graag willen veranderen. Ik geniet ook van dingen die mij helpen groeien als persoon en de wereld een betere plek maken.

Toen ik voor het eerst begon met programmeren, was dat in HTML. Sommige mensen beschouwen dat niet als een programmeertaal, en dat kan waar zijn, maar HTML introduceerde mij in het concept van programmeren. Tegenwoordig weet ik een aardig beetje over HTML, CSS, JS en Python.

Vergeleken met andere programmeurs weet ik misschien niet veel, maar ik hou er niet van om mezelf met anderen te vergelijken. Vaak, wanneer ik codeer, leef ik volgens de regel: "Mijn code werkt niet, ik weet niet waarom. Ik verander iets, nu werkt het, maar ik weet nog steeds niet waarom." Ik probeer steeds meer de code die ik zelf schrijf of de code die ik kopieer te begrijpen.

Momenteel ben ik een student en volg ik het (Creatieve) Software Developer programma aan het Grafisch Lyceum Rotterdam. Dit programma duurt drie jaar. Ik begon in augustus 2024, dus ik zal klaar zijn in 2027, mits ik niet achter raak. Daarna zou ik graag een Cyber Security programma willen volgen op een hoger beroepsonderwijs (HBO) niveau.

Ik ben een kattenmens. Veel katten zijn erg vriendelijk en schattig. Ik heb zelf ook katten gehad. Hieronder kun je twee foto's zien van een van die katten.

Ik heb een zus, en op het moment dat ik dit schrijf, is ze 19 jaar oud (jong). Ze is transgender, en ik heb daar geen probleem mee. Als jij dat wel hebt, kan het me heel weinig schelen. Ze heeft ook haar eigen website; hier kun je die bekijken. Ze volgt hetzelfde programma als ik, (Creatieve) Software Developer, maar we zitten niet in dezelfde klas.
